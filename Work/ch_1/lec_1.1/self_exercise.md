# Self Exercise - 1.1

## Q1. What is Machine Learning, and how does it differ from traditional programming?

Machine Learning (ML) is a branch of Artificial Intelligence where computers learn patterns from data and improve performance without explicit programming.

### Traditional Programming vs Machine Learning

| Traditional Programming | Machine Learning |
| --- | --- |
| Data + Rules → Output | Data + Examples → Learned Rules |
| Rules are manually written | Rules are learned automatically |

1. Supervised Learning  
2. Unsupervised Learning  
3. Reinforcement Learning  
4. Semi-Supervised Learning

## Q3. Why is data important in Machine Learning?

Data is the foundation of Machine Learning because ML models learn patterns from data.

---

## Q4. What role do patterns play in training machine learning models?

Patterns help models identify relationships and make predictions on unseen data.

---

## Q5. What is Artificial Intelligence (AI)?

Artificial Intelligence (AI) is the field of creating intelligent systems capable of performing tasks requiring human intelligence.

---

## Q6. How is Machine Learning a subset of Artificial Intelligence?

AI is the broader concept, while ML is a subset that allows systems to learn from data.

---

## Q7. How does Deep Learning differ from traditional Machine Learning?

Deep Learning uses neural networks with many layers and automatically learns features from data.

---

## Q8. Why does Deep Learning require large amounts of data and computational power?

Deep Learning models contain millions of parameters and require GPUs and large datasets for training.

---
